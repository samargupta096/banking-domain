## General Banking

| Term | Definition |
|------|-----------|
| **ADI** | Authorised Deposit-taking Institution — a bank licensed by APRA to accept deposits |
| **Bancassurance** | Distribution of insurance products through a bank's sales channels |
| **Basis Point (bp)** | 1/100th of a percentage point. So 25bps = 0.25%. Used to express interest rate changes |
| **Big 4** | Australia's four largest banks: CBA, Westpac, NAB, ANZ |
| **Book Value** | The net asset value of a company (assets minus liabilities) as shown on its balance sheet |
| **CAGR** | Compound Annual Growth Rate — average annual growth rate over a period |
| **Central Bank** | A country's primary monetary authority (e.g., RBA in Australia, Fed in US) |
| **Counterparty** | The other party in a financial transaction |
| **D-SIB** | Domestic Systemically Important Bank — banks whose failure would harm the entire financial system |
| **FI** | Financial Institution — any entity that deals in financial transactions |
| **Fintech** | Financial Technology — technology-driven company providing financial services |
| **FTE** | Full-Time Equivalent — standardized measure of employee headcount |
| **GDP** | Gross Domestic Product — total value of goods and services produced in a country |
| **Intermediation** | The core banking function of connecting savers with borrowers |
| **Leverage** | Using borrowed money to amplify returns; also, a ratio of debt to equity |
| **Moral Hazard** | Risk that a party insulated from risk behaves differently (e.g., "too big to fail") |
| **NIM** | Net Interest Margin — (Interest Income − Interest Expense) / Average Earning Assets |
| **NII** | Net Interest Income — the dollar amount earned from the interest rate spread |
| **ROE** | Return on Equity — Net Income / Shareholders' Equity. Key profitability metric for banks |
| **ROA** | Return on Assets — Net Income / Total Assets |
| **Spread** | The difference between two interest rates (e.g., lending rate minus funding rate) |
| **Systemic Risk** | Risk that failure of one institution triggers cascading failures across the financial system |
| **Too Big to Fail** | A bank so large that its failure would cause a financial crisis |
| **Universal Bank** | A bank operating across retail, commercial, and investment banking |
| **Yield Curve** | Graph showing interest rates across different maturities |

## Deposits & Accounts

| Term | Definition |
|------|-----------|
| **BSB** | Bank-State-Branch number — 6-digit code identifying a bank branch in Australia |
| **CIF** | Customer Information File — central record of customer data in a bank's system |
| **Current Account** | A transaction account allowing unlimited deposits and withdrawals (also called checking account) |
| **Demand Deposit** | Deposit that can be withdrawn at any time without notice |
| **Dormant Account** | An account with no customer-initiated activity for a specified period (typically 12+ months) |
| **IBAN** | International Bank Account Number — standardized international account identifier (not used in Australia) |
| **Minimum Balance** | The lowest balance that must be maintained in an account to avoid fees or earn interest |
| **Notice Deposit** | Deposit requiring advance notice (e.g., 31 days) before withdrawal |
| **Offset Account** | A savings account linked to a home loan; the balance reduces the mortgage interest charged |
| **Overdraft** | A facility allowing an account to go into a negative balance up to an agreed limit |
| **Savings Account** | An interest-bearing account designed for accumulating funds |
| **Term Deposit** | A fixed-term investment at a set interest rate, with penalties for early withdrawal |
| **Transaction Account** | An account for everyday banking — payments, receipts, direct debits |
| **Unbanked** | People who do not have a bank account |
| **Underbanked** | People with a bank account but limited access to financial services |

## Lending & Credit

| Term | Definition |
|------|-----------|
| **Amortization** | The gradual repayment of a loan through scheduled payments of principal and interest |
| **APR** | Annual Percentage Rate — the total yearly cost of a loan including fees, expressed as a rate |
| **Arrears** | Overdue payments — a borrower "in arrears" has missed one or more repayments |
| **Balloon Payment** | A large final payment due at the end of a loan term |
| **Bridging Loan** | A short-term loan used to "bridge" the gap between buying a new property and selling an existing one |
| **Collateral** | An asset pledged by a borrower as security for a loan |
| **Comparison Rate** | An Australian requirement — a rate that includes both the interest rate and standard fees |
| **Covenant** | A condition in a loan agreement the borrower must satisfy (e.g., maintaining a debt ratio) |
| **Credit Bureau** | An organization that collects and provides credit information (Equifax, Illion, Experian in AU) |
| **Credit Limit** | The maximum amount a borrower can access on a revolving credit facility |
| **Credit Score** | A numerical score representing a borrower's creditworthiness |
| **Cross-Collateralization** | Using one property as security for multiple loans |
| **Debt-to-Income (DTI)** | Total debt divided by gross income — used to assess repayment capacity |
| **Default** | Failure to meet the terms of a loan agreement, typically non-payment for 90+ days |
| **Discharge** | The formal release of a mortgage after the loan is fully repaid |
| **DSR** | Debt Service Ratio — annual debt payments divided by annual income |
| **ECL** | Expected Credit Loss — forward-looking estimate of potential losses on a loan portfolio |
| **Equity (Property)** | The difference between a property's market value and the outstanding mortgage |
| **Fixed Rate** | An interest rate that remains constant for a specified period |
| **Forbearance** | Temporary modification of loan terms due to borrower hardship |
| **Foreclosure** | Legal process where a lender takes possession of a property due to default (called "mortgagee sale" in AU) |
| **Guarantor** | A person who agrees to repay a loan if the primary borrower defaults |
| **Hardship** | A program allowing borrowers facing genuine difficulty to negotiate modified repayment terms |
| **Interest Only (IO)** | A repayment type where only interest is paid — the principal doesn't reduce |
| **LMI** | Lenders Mortgage Insurance — insurance protecting the lender if LVR exceeds 80% (borrower pays the premium) |
| **LVR** | Loan-to-Value Ratio — loan amount as a percentage of the property value |
| **Mortgage** | A loan secured against real property |
| **Negative Gearing** | When the cost of an investment property (interest, expenses) exceeds the rental income — the loss is tax-deductible in AU |
| **NPL** | Non-Performing Loan — a loan in default or close to default (90+ days past due) |
| **P&I** | Principal and Interest — repayment type where both the loan amount and interest are paid |
| **Pre-Approval** | Conditional approval for a loan amount before selecting a property |
| **Principal** | The original amount borrowed (excluding interest) |
| **Redraw** | A facility allowing a borrower to access extra repayments they've made on their loan |
| **Refinance** | Replacing an existing loan with a new one, often for a better rate or different terms |
| **Secured Loan** | A loan backed by collateral (e.g., car loan, mortgage) |
| **Serviceability** | The ability of a borrower to meet repayment obligations |
| **Settlement (Property)** | The legal process of transferring property ownership and exchanging funds |
| **Split Loan** | A loan divided into two portions — part fixed rate and part variable rate |
| **Subordination** | The ranking of debt priority — senior debt is repaid before subordinated debt |
| **Unsecured Loan** | A loan not backed by collateral (e.g., personal loan, credit card) |
| **Variable Rate** | An interest rate that changes in line with market conditions |
| **Write-Off** | Removing a loan from the bank's balance sheet when it's deemed unrecoverable |

## Payments & Transfers

| Term | Definition |
|------|-----------|
| **Acquirer** | The bank/payment processor that handles card payments on behalf of a merchant |
| **Authorization** | Real-time approval/decline of a payment transaction |
| **BECS** | Bulk Electronic Clearing System — Australia's batch payment system for direct credits/debits |
| **BIC** | Bank Identifier Code — 8 or 11-character code identifying a bank on the SWIFT network |
| **BSB** | Bank-State-Branch — 6-digit identifier for Australian bank accounts |
| **Chargeback** | Reversal of a card transaction initiated by the cardholder's bank |
| **Clearing** | The process of transmitting, reconciling, and confirming payment instructions between banks |
| **Contactless** | NFC-based payment method ("tap and go") |
| **Correspondent Bank** | A bank that provides services on behalf of another bank, typically for international payments |
| **Credit Transfer** | A "push" payment — the payer initiates sending money to the payee |
| **DDR** | Direct Debit Request — authorization for a merchant/organization to debit a customer's account |
| **Direct Credit** | A payment pushed from a payer's account to a payee's account (e.g., salary) |
| **Direct Debit** | A payment pulled from a payer's account by an authorized payee (e.g., subscription) |
| **EMV** | Europay, Mastercard, Visa — the chip-based card technology standard |
| **Finality** | The point at which a payment becomes irrevocable |
| **Interchange Fee** | Fee paid by the acquirer to the issuer in a card transaction |
| **Issuer** | The bank that issued the payment card to the cardholder |
| **ISO 20022** | Universal financial messaging standard — XML/JSON-based, rich data |
| **Loro Account** | "Their account with them" — a third party's perspective on a Nostro/Vostro relationship |
| **MSF** | Merchant Service Fee — total fee charged to a merchant for accepting card payments |
| **Nostro Account** | "Our account with you" — a bank's account held at a correspondent bank in a foreign currency |
| **NPP** | New Payments Platform — Australia's real-time payment infrastructure |
| **Off-us** | A transaction where the payer and payee bank with different institutions |
| **On-us** | A transaction where both parties bank with the same institution |
| **Osko** | An overlay service on NPP for basic fast payments |
| **PayID** | An alias (phone, email, ABN) linked to a bank account for NPP payments |
| **PayTo** | NPP overlay service for mandated/recurring payments (replaces DDR agreements) |
| **POS** | Point of Sale — the terminal where card payments are accepted |
| **RITS** | Reserve Bank Information and Transfer System — Australia's high-value settlement system |
| **RTGS** | Real-Time Gross Settlement — individual payments settled in real-time |
| **Scheme** | The card network (Visa, Mastercard, eftpos) |
| **Settlement** | The actual transfer of money between banks to complete a payment |
| **SWIFT** | Society for Worldwide Interbank Financial Telecommunication — global financial messaging network |
| **Tokenization** | Replacing a card number with a unique device-specific token for security |
| **Vostro Account** | "Your account with us" — a correspondent bank's term for another bank's account held with them |

## Risk Management

| Term | Definition |
|------|-----------|
| **Concentration Risk** | Risk from excessive exposure to a single borrower, industry, or region |
| **Counterparty Risk** | Risk that the other party in a transaction will fail to fulfill their obligations |
| **Credit Risk** | Risk of loss from a borrower or counterparty failing to repay |
| **EAD** | Exposure at Default — the total amount owed at the time of default |
| **Expected Loss** | PD × LGD × EAD — the "normal" level of loss a bank plans for |
| **Hedging** | Using financial instruments to offset risk (e.g., interest rate swap to fix a floating rate) |
| **Interest Rate Risk** | Risk of loss from changes in interest rates |
| **LGD** | Loss Given Default — the percentage of exposure lost when a borrower defaults |
| **Liquidity Risk** | Risk that a bank cannot meet its financial obligations as they come due |
| **Market Risk** | Risk of loss from changes in market prices (interest rates, FX, equities) |
| **Operational Risk** | Risk of loss from inadequate/failed processes, people, systems, or external events |
| **PD** | Probability of Default — the likelihood that a borrower will default |
| **Provisioning** | Setting aside funds to cover expected losses on loans |
| **Reputational Risk** | Risk of damage to a bank's reputation affecting business and profitability |
| **Risk Appetite** | The amount and type of risk a bank is willing to accept to achieve its objectives |
| **Stress Testing** | Simulating extreme economic scenarios to test a bank's resilience |
| **Unexpected Loss** | Losses beyond what is expected — the purpose of capital buffers |
| **VaR** | Value at Risk — maximum expected loss over a given period at a given confidence level |

## Compliance & Regulatory

| Term | Definition |
|------|-----------|
| **AML** | Anti-Money Laundering — laws and procedures to prevent money laundering |
| **AUSTRAC** | Australian Transaction Reports and Analysis Centre — Australia's financial intelligence agency |
| **CDD** | Customer Due Diligence — process of verifying customer identity and assessing risk |
| **Compliance** | The function ensuring a bank adheres to laws, regulations, and internal policies |
| **CTF** | Counter-Terrorism Financing — measures to prevent financing of terrorism |
| **EDD** | Enhanced Due Diligence — deeper investigation for higher-risk customers |
| **FAR** | Financial Accountability Regime — Australian legislation holding senior executives individually accountable |
| **FATF** | Financial Action Task Force — international body setting AML/CTF standards |
| **IFTI** | International Fund Transfer Instruction — report filed with AUSTRAC for international transfers |
| **KYC** | Know Your Customer — the process of verifying a customer's identity before and during business |
| **PCI-DSS** | Payment Card Industry Data Security Standard — security standard for card data |
| **PEP** | Politically Exposed Person — individuals in prominent public positions requiring enhanced scrutiny |
| **Sanctions** | Restrictions on financial dealings with specific countries, entities, or individuals |
| **SAR/SMR** | Suspicious Activity Report / Suspicious Matter Report — report filed when suspicious activity is detected |
| **SOX** | Sarbanes-Oxley Act — US legislation on financial reporting controls (relevant for US-listed entities) |
| **Three Lines of Defence** | Governance model: 1st (business), 2nd (risk/compliance), 3rd (internal audit) |
| **TTR** | Threshold Transaction Report — report filed for cash transactions of $10,000+ |
| **UBO** | Ultimate Beneficial Owner — the individual person who ultimately owns or controls an entity |

## Capital & Basel

| Term | Definition |
|------|-----------|
| **AT1** | Additional Tier 1 capital — hybrid instruments like preference shares |
| **Basel III/IV** | International banking regulation framework setting minimum capital and liquidity standards |
| **CAR** | Capital Adequacy Ratio — total regulatory capital divided by risk-weighted assets |
| **CCB** | Capital Conservation Buffer — additional 2.5% CET1 above the minimum |
| **CCyB** | Countercyclical Capital Buffer — additional buffer that varies with economic conditions |
| **CET1** | Common Equity Tier 1 — highest quality capital (shares + retained earnings) |
| **ICAAP** | Internal Capital Adequacy Assessment Process — a bank's own assessment of capital needs |
| **LCR** | Liquidity Coverage Ratio — liquid assets vs. net cash outflows over 30 days |
| **Leverage Ratio** | Tier 1 capital to total (non-risk-weighted) exposure |
| **NSFR** | Net Stable Funding Ratio — available stable funding vs. required stable funding |
| **Pillar 1** | Minimum capital requirements (Basel framework) |
| **Pillar 2** | Supervisory review process (Basel framework) |
| **Pillar 3** | Market discipline through disclosure (Basel framework) |
| **RWA** | Risk-Weighted Assets — total assets adjusted for riskiness |
| **Tier 1 Capital** | Core capital: CET1 + AT1 |
| **Tier 2 Capital** | Supplementary capital: subordinated debt, general provisions |

## Investment & Markets

| Term | Definition |
|------|-----------|
| **AUM/FUM** | Assets/Funds Under Management — total value of investments managed on behalf of clients |
| **Bond** | A fixed-income security — essentially a loan to the issuer (government or corporate) |
| **Coupon** | The interest payment on a bond |
| **DCM** | Debt Capital Markets — the division that helps raise debt financing |
| **Derivative** | A financial instrument whose value derives from an underlying asset (options, futures, swaps) |
| **Dividend** | A distribution of company profits to shareholders |
| **ECM** | Equity Capital Markets — the division that helps raise equity financing (IPOs, etc.) |
| **ETF** | Exchange-Traded Fund — a fund traded on a stock exchange that tracks an index or basket |
| **Franking Credits** | Tax credits attached to dividends in Australia (imputation system) |
| **IPO** | Initial Public Offering — the first sale of a company's shares to the public |
| **LC** | Letter of Credit — bank guarantee of payment used in trade finance |
| **M&A** | Mergers and Acquisitions — corporate transactions to buy, sell, or combine companies |
| **Market Making** | Providing continuous buy and sell quotes for a financial instrument |
| **Securitization** | Pooling financial assets (e.g., mortgages) into tradeable securities (RMBS, ABS) |
| **SMSF** | Self-Managed Superannuation Fund — a member-directed retirement fund |
| **Superannuation (Super)** | Australia's compulsory retirement savings system |
| **Swap** | An agreement to exchange cash flows — e.g., fixed vs floating interest rate |
| **Underwriting** | Guaranteeing the sale of securities — the bank takes on risk of unsold portions |
| **Yield** | The return on an investment, typically expressed as an annual percentage |

## Technology

| Term | Definition |
|------|-----------|
| **API** | Application Programming Interface — a way for systems to communicate |
| **BaaS** | Banking-as-a-Service — bank infrastructure offered via APIs to non-banks |
| **BCP** | Business Continuity Plan — procedures for maintaining operations during disruptions |
| **CAB** | Change Advisory Board — committee that approves changes to production systems |
| **CBS** | Core Banking System — the primary system for account management and transaction processing |
| **CDC** | Change Data Capture — a technique for real-time data replication from databases |
| **CDR** | Consumer Data Right — Australia's open data framework (Open Banking) |
| **CQRS** | Command Query Responsibility Segregation — pattern separating read and write operations |
| **CPS 234** | APRA prudential standard for Information Security |
| **DR** | Disaster Recovery — systems and procedures for recovering from major outages |
| **EOD** | End of Day — the nightly batch processing cycle |
| **ESB** | Enterprise Service Bus — middleware for integrating enterprise applications |
| **FAPI** | Financial-grade API — enhanced security standard for financial APIs |
| **GL** | General Ledger — the master financial record of all transactions |
| **ITSM** | IT Service Management — framework for managing IT services (e.g., ITIL) |
| **mTLS** | Mutual TLS — both client and server authenticate each other with certificates |
| **PVT** | Performance and Volume Testing — testing system behavior under load |
| **RTO** | Recovery Time Objective — maximum acceptable downtime |
| **RPO** | Recovery Point Objective — maximum acceptable data loss (in time) |
| **SIT** | System Integration Testing — testing interactions between systems |
| **SLA** | Service Level Agreement — formal agreement on system availability/performance |
| **SOA** | Service-Oriented Architecture — architectural pattern using discrete services |
| **SoE** | System of Engagement — customer-facing systems |
| **SoR** | System of Record — authoritative data source |
| **STP** | Straight-Through Processing — fully automated transaction processing with no manual intervention |
| **UAT** | User Acceptance Testing — business users validate the system meets requirements |

## Australian Banking Specific

| **FAR** | Financial Accountability Regime — individual accountability for senior bank executives |
| **FCS** | Financial Claims Scheme — guarantees deposits up to $250,000 per ADI per depositor |

| **Royal Commission** | 2017-2019 inquiry into banking misconduct — led to sweeping reforms |
